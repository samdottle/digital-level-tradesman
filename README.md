# Digital Level Tradesman — Web App + Firmware

**Version 1.9** · **Firmware version 2.5** · 2026-07-19 · MPU-6050 · Arduino Nano ESP32 · BLE Nordic UART

## What changed in v1.9
Added a vertical line through the center of the bubble, reaching the
bubble's own top and bottom edge with no gap. Previously the bubble was
a plain filled shape with nothing marking its own center, so lining it
up against the tube's fixed center tick meant eyeballing the middle of a
rounded shape against a thin line. Now it's a precise line-to-line match.
