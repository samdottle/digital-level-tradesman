# Digital Level Tradesman — Web App + Firmware

**Version 1.8** · **Firmware version 2.5** · 2026-07-19 · MPU-6050 · Arduino Nano ESP32 · BLE Nordic UART

## What changed in v1.8
Two visibility fixes, no functional changes:

1. **Added a version-only subtitle back under the header title** ("v1.8"),
   so the running version is visible at a glance without scrolling to the
   footer. The v1.6 removal was specifically of the chip/sensor detail
   ("MPU-6050 · ESP32 · BLE"), not the version number — this restores
   just the version, without the chip/sensor info.

2. **Removed "Arduino Nano ESP32 + MPU-6050 · BLE Nordic UART" from the
   footer** — v1.6's chip/sensor removal only ever covered the header;
   this line in the footer was missed at the time.

## Where to check the version now
Right under the product name at the top of the screen — no more need to
scroll to the footer.
